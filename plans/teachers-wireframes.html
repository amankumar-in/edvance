API Calls

GET /api/tasks/{id}
GET /api/tasks?assignedTo=class:{classId}&status=pending_approval
POST /api/tasks/{id}/review (approve/reject)

Review Process

Approve: Task is marked as approved, points are awarded
Reject: Task is returned to student for revision
Feedback: Optional comments for the student
Points: Can adjust the points awarded (partial credit)

Points Transaction

When a task is approved, a point transaction is automatically created:
  • source: "task"
  • sourceId: ID of the task
  • amount: Points awarded
  • type: "earned"
  • awardedBy: Teacher ID
  • awardedByRole: "teacher"
