Analytics Types

School administrators can view various analytics:

  • Attendance: Detailed attendance analysis by grade, class, and time period

  • Tasks: Task completion rates, popular task types, challenging tasks
  • Grades: Academic performance comparisons and trends

Export Options

Reports can be exported in various formats:
  • PDF (for sharing with stakeholders)
  • CSV (for further analysis)
  • Excel (for detailed data manipulation)
  • Email (send to district administration)

Point System Configuration

School administrators can customize:

  • Attendance point values and streak bonuses

Best Practices

Steady Progression: Create attainable levels with reasonable gaps
Multiple Pathways: Allow students to earn points through various activities
Meaningful Rewards: Ensure redemption options are motivating and appropriate
